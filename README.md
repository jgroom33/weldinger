# Weldinger — Durango, CO

Static, single-page site for a custom home metalwork studio based in Durango, Colorado. Built with semantic HTML and vanilla CSS—ready for GitHub Pages.

## Local preview
Open `index.html` in your browser. No build steps required.

## Deploy on GitHub Pages
1. Commit these files and push to the default branch (often `main`).
2. In the repo settings, enable GitHub Pages with source set to `Deploy from a branch` → branch `main` → folder `/ (root)`.
3. Save. GitHub will publish `index.html` automatically.
